# Print-Shop-Management-System
A full-stack web app for managing a school print shop — student accounts, charges, payments, batch billing, print job tracking, and configurable pricing.

Stack: Flask + SQLite

Features

Student management (add/edit/delete/search) organized by batch and stream
Transaction ledger — balances always calculated as SUM(transactions), never stored directly
Batch charges for entire groups at once, plus individual student charges/payments
Print job tracking with optional auto-charging based on configurable prices
Dashboard with daily stats and outstanding balances

Built as a learning project to practice database design (3NF schema) and full-stack development.
